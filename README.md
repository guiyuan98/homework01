# 个人数字名片项目 📇

这是我的个人数字名片网页，展示了我的个人基本信息、自我介绍、技术栈以及兴趣爱好等信息。本项目作为 Web 开发基础实验作品，旨在练习 HTML 结构化布局与 CSS 样式美化。

## 🌐 访问链接

**[点击此处访问我的个人名片预览页](https://gitee.com/guiyuan98/homework01)**

## 🛠️ 技术实现

- **HTML5**: 构建网页的语义化结构。
- **CSS3**: 实现响应式布局、渐变背景、阴影效果及交互动画。
- **Dicebear API**: 用于生成个性化的卡通头像。

## 📂 文件结构

- `index.html`: 网页的主体结构文件。
- `style.css`: 网页的样式美化文件。
- `README.md`: 项目的项目说明文档。
